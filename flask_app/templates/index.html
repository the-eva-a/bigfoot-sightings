Bigfoot Sightings API

Explore the wonders of Bigfoot sightings across the USA!

About the API

The Bigfoot Sightings API provides detailed information about reported Bigfoot sightings. Search by location, time period, keywords, and proximity to specific areas.

Available Endpoints

Endpoint Description
/ Homepage with information about the API.
/info/<int:report_number> Retrieve detailed information about a specific report.
/state-reports?state=<state_name> Get all reports for a specific state (case-insensitive).
/county-reports?county=<county_name>&state=<state_name> Retrieve all reports from a specific county, with an optional state filter.
/nearby-sightings?location=<location>&radius=<radius> Find sightings within a specific radius of a location.
/time-period-search?start_year=<year>&end_year=<year> Search sightings by date range, with optional state or county filters.
/word-search?keyword=<keyword> Search for sightings containing specific keywords in their descriptions.

Example Usage

To search for sightings in Washington State, try:

http://127.0.0.1:5000/state-reports?state=Washington
